# Meshtastic BLE Controller
This repo is for a custom BLE controller for Meshtastic devices. 
It allows users to connect to their Meshtastic devices via Bluetooth and control various functions such as sending messages, changing settings, and monitoring device status.
The app allows loading the device keys to easily connect to multiple devices.

## Features
- Connect to Meshtastic devices via BLE
- Send and receive messages
- Change device settings (e.g., channel, power level)
- Monitor device status (e.g., battery level, signal strength)
- User-friendly interface
